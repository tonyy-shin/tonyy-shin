CS student at RIT and Technology Intern at EY Korea, building Python automation tools for financial workflows.

### Projects

**[Smart Sweat-Band](https://github.com/tonyy-shin/Smart-Sweat-Band-Project)** *(in progress — Phase 1: Sensor Fusion)* — Funded wearable prototype built at RIT on an ESP32-S3 for post-exercise recovery monitoring. All three sensors now run concurrently on one board: a MAX30205 clinical thermometer and Grove GSR electrodes, plus an SHT45 humidity sensor housed in a custom ePTFE vapor chamber that passes water vapor while blocking liquid sweat. A planned companion app will turn these streams into a Recovery Readiness Score with personalized rehydration prescriptions and thermal-regulation alerts.

**[cost-allocation](https://github.com/tonyy-shin/cost-allocation)** — Desktop app built for EY Korea automating common cost allocation across cost centers. Multi-cycle pandas allocation engine with tkinter GUI, packaged as a Windows EXE for non-technical finance users.

**Also**
- **[voucher-analysis-automation](https://github.com/tonyy-shin/voucher-analysis-automation)** — Excel-to-PDF pipeline generating branch attribution reports from expense voucher data for EY Korea's FS Assurance team.
- **[Draft-Optimized FPL Predictor](https://github.com/tonyy-shin/pl-predictor-project)** *(in progress — data pipeline)* — Automated data pipeline, statistical model, and Streamlit dashboard for the Fantasy Premier League Draft. Designed to pair Dixon-Coles Poisson modeling (with time-decay) with MILP squad optimization over live FPL Draft API availability to maximize expected points under formation and club-limit constraints. Currently building out data ingestion; built with pandas, SciPy, SQLite, and Streamlit.
